Introduction

CSS provides style and layout to a web page. Cascading Style Sheets also known as (CSS)can be simple yet effective.
CSS can not be used alone, it is presentation of a document in a markup language such as HTML.

CSS is designed to enable the separation of presentation and content, including layout, colors, and fonts.

Text Color

When wanting to add text color to your HTML information. You will have to use the tag,


h1{ color:_____;}
  • Then pick the color of your choice.
  • When adding color always end to semi-colons.

Background Color

When choosing background, you would use the same elements as when choosing Text color but instead of using the tag "color". You would use "background-color".

h1 {background-color:________;}
  • Then pick the color of your choice.
  • End with semi-colon.
